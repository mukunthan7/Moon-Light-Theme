# Moon-Light-Theme

Moon Light theme is a dark material-like theme with blue tones.

## Installation

1.  Open Extensions on the sidebar
2.  Search for "**Moon Light**"
3.  Install it
4.  Reload the editor

# Mac

5.  Press ⌘K + ⌘T

# Windows and Linux

6.  Press Ctrl K+Ctrl T
7.  Pick "**Moon Light**" one from the list and hit enter
8.  Enjoy!
